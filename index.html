MetaScan: A Multi-Signal Static Malware Triage System with Zero False Positives for SOC Environments
role: Undergraduate Researcher  ·  2025-09 → present  ·  Punjab, India
96.32% accuracy 1.00 precision 83.33% recall 0 false positives F1: 0.91

Designing and evaluating MetaScan, a lightweight static malware triage framework combining Shannon entropy analysis, YARA-based pattern detection, file-type awareness, and heuristic scoring — deployed as a REST API for real-time SOC use. Evaluated on 1,357 curated samples (1,057 benign / 300 malicious).

  • Demonstrated multi-signal static analysis significantly outperforms entropy-only approaches on benign high-entropy files.
  • Introduced an "unknown" classification tier for controlled analyst escalation without sacrificing precision.
  • Deployed as REST API (Flask + Gunicorn) for direct integration into automated SOC pipelines.
  • Zero false positives on evaluation set — no benign file misclassified as malicious.
